PHP Developer task

Task
Create a basic registration & login system.
Users can view articles from the list (articles can be taken from any public URL - https://github.com/public-apis/public-apis?tab=readme-ov-file#news
)
Near each article there should be an option to hide the article.
Each article can be added as a favorite by pressing the star near the article.
Favorite articles are ordered at the very top with the latest favorite being the first one.
There should be at least 3 feature/integration tests written.

Technology
You can use any stack as long as the instructions are clear on how it can be run locally.
Additional points for using Laravel and Livewire.
Databases can be either MySQL but for simplicity SQLite is acceptable.
